## 🔥 Quick Start: OHLCV History & Backtesting

The primary way to get OHLCV data for backtesting is via `vopr/data_ingestion.py`:

```python
from vopr.data_ingestion import fetch_ohlcv, get_spot_price, get_available_expirations

# Get 90 days of daily OHLCV (Tradier first, yfinance fallback)
df = fetch_ohlcv("TSLA", days=90)
# Returns: pd.DataFrame with DatetimeIndex, columns: Open, High, Low, Close, Volume

# Get spot price
price = get_spot_price("TSLA")

# Get options expirations
expiries = get_available_expirations("TSLA")
```

**API endpoint** (from Venus:8100):

```bash
# OHLCV history via API
curl "http://192.168.2.172:8100/api/vopr/history?ticker=TSLA&days=90"
# Returns JSON array: [{date, open, high, low, close, volume}, ...]
```

## ⚠️ Critical Gotchas for Cross-Repo Work

1. **deep_dive.* files are sacred** — NEVER delete or gitignore them. They cost money to generate.
2. **Picks cache is 5 min** — alpha-momentum's `/api/picks/today` has a 5-min TTL cache from GH Pages.
3. **Tickers have two types of pages**: dossier ticker pages (free, daily) and deep dives (expensive AI, one-time). Don't confuse them.
4. **GitHub Actions** — `watchlist.txt` changes trigger deep dive generation via `.github/workflows/watchlist_dive.yml`. Don't accidentally trigger this by modifying the file programmatically.
5. **.env files are NOT synced** — mphinance stuff lives on Vultr/GitHub, alpha-momentum lives on Venus. Different machines, different secrets.
