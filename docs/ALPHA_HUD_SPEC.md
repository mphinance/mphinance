## Color System (3-Color)

| Variable | Hex | Usage |
|----------|-----|-------|
| `--green` | `#00ff41` | Bullish, up, undervalued, buying, oversold RSI |
| `--red` | `#f0b400` | Bearish, down, caution — **warm gold, NOT red** |
| `--danger` | `#e53935` | OVERVALUED, overbought RSI >70, puts bar, selling |
| `--blue` | `#00f3ff` | IV/HV, panel accent, interactive elements |
| `--amber` | `#ffb000` | Panel titles, grade pills |
| `--purple` | `#a855f7` | AI synthesis accent, branding |

**MA values**: green if price > MA, gold if price < MA
**EMA stack**: green if BULLISH, gold if BEARISH, amber if PARTIAL
**Score boxes**: green if >60, gold if ≤60

```html
<iframe src=".../widgets/signal-bar.html?t=TSLA" width="700" height="160" ...></iframe>
<iframe src=".../widgets/valuation.html?t=TSLA" width="340" height="280" ...></iframe>
```

All widgets accept `?t=TICKER` query param. Data auto-fetches from GH Pages.

## Porting to TraderDaddy / Alpha-Momentum

Key considerations for porting to a React/Next.js dashboard:

1. **Data source**: Replace GH Pages JSON fetch with your FastAPI `/api/v1/ticker/{t}` endpoint
2. **Components**: Each panel (Signal Strip, Valuation, AI Synthesis, Options Flow, Institutional, MAs, Pivots) maps to a React component
3. **Styling**: Use the CSS variable system above with Frosted UI glass effects
4. **Color logic**: Same 3-color rules, use CSS custom properties
5. **Share**: `html2canvas` → Web Share API (already implemented, just port the JS)

### Component breakdown for React

- `<TickerHeader ticker={data} />`
- `<SignalStrip signals={data} />`
- `<ValuationPanel valuation={data.valuation} />`
- `<AISynthesis scores={data.scores} analysis={data.ai_analysis} />`
- `<OptionsFlow volumeStats={data.volumeStats} />`
- `<InstitutionalIntel tickertrace={data.tickertrace} />`
- `<MovingAverages ema={data.technical_analysis.ema} trend={data.technical_analysis.trend} price={data.currentPrice} />`
- `<PivotsOscillators pivots={data.technical_analysis.pivots} oscillators={data.technical_analysis.oscillators} />`
